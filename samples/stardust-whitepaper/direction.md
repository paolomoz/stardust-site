# Direction

User phrase: **clean, playful, bold**.

Restated in Stardust dimensions:
- **Distinctiveness:** familiar → distinctive (lean into Stardust's singular palette + mono-tagged structure rather than generic SaaS landing chrome)
- **Density:** generous (editorial, not commerce-grid)
- **Mode:** brand-faithful — palette and type pinned to Stardust's identity surface
- **Register:** brand · technical (the doc is a strategy memo, not a marketing pitch)
- **Ground:** ink-deep dominant, with dust paper used as breathing room

Three variants explore one axis each.

## Variant A · Clean
**Thesis:** quiet authority. The brief reads as an editorial memo — generous whitespace, modular type, single amber accent per scene, dust particles only in the hero. Restrained motion. The reader's eye finds the structure without the page raising its voice.

**Movements:**
- Generous vertical rhythm — 110–140px section padding
- Type scale modular 1.25, max body 16px, max headline 64px
- Amber used only for eyebrows + a single highlighted phrase per section
- Ground: ink-deep top → dust paper for sections 03–05 → ink-deep return for closing
- Motion: scroll-driven fade-in only, no hover choreography

## Variant B · Playful
**Thesis:** lively craft. The brief breaks its grid intentionally — varying section widths, color rotation between scenes (ink → ink-soft → dust → ink), expressive type pairings (display + italic + mono in close proximity), micro-interactions (eyebrow numbers tick, hero diagram reveals on scroll, logo strip shimmers).

**Movements:**
- Section ground rotates: every section a different surface color
- Display type alternates roman / italic mid-headline
- Mono code chips at section corners (`brief + seed = star` everywhere)
- Architecture diagram becomes interactive — hover lights up each node
- Cursor-following dust particle near hero
- Logo strip enters with staggered y-shift

## Variant C · Bold
**Thesis:** dramatic claim. Oversized hero (124pt+ display, full-bleed gradient ground), full-bleed before/after with edge-to-edge plane image, scroll-driven section transitions where headlines slide in from the gutter, sections snap, and the architecture diagram becomes a hero scene of its own with Slicc as a luminous band over Stardust → Snowflake → AEM.

**Movements:**
- Hero: 124pt display, dust-particle overlay, ground gradient
- Before/after: 100vw, no inner padding, frames touch the page edge
- Section headlines slide in from left on scroll
- Architecture: full-width, ink-soft block with amber connector lines
- "Why this matters for AEM" rendered as 3 oversized stacked claims, each a scene
- Footer: full-width amber rule on top, mono links

## What every variant must keep

- Stardust palette — ink/amber/dust only
- SF Pro Display + SF Mono
- Numbered mono eyebrows
- Voice principles: plain words, surfaced mechanism, short sentences
- Virgin Atlantic before/after as the proof point
- 8 brand logos in the strip
- All 7 sections of brief content + footer links
